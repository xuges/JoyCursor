# JoyCursor

**Joystick controls the mouse cursor!**

[中文](doc/README_cn.md)



## Support

**OS:** Windows11 x64

**Device:** XBOX-Like Joystick



## Usage

Double-click the `JoyCursor.exe` file. 

![](doc/icon.png)

Normally, the icon  will appear in the taskbar, and you can use your joystick control the mouse cursor now.
